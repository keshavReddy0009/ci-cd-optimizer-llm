CI/CD Pipeline Optimizer

This is a simple Flask app that is part of the CI/CD YAML Optimizer using LLMs project.

The backend:

  • Serves this HTML page using Flask
  • Exposes a health check at /health
  • Is tested and checked via GitHub Actions

The project also includes a GitHub Actions workflow that uses a Large Language Model to analyze CI/CD YAML files for performance, cost, and security improvements.
